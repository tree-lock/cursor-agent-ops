# Cursor 配置说明

本目录存放 Cursor 编辑器的项目级配置，用于为 AI 提供持久化指导。

## 结构

- **`rules/`**：规则文件（`.mdc`），在对话中自动注入上下文。
  - 按 **文件匹配**（`globs`）或 **始终应用**（`alwaysApply: true`）生效。
  - 建议每条规则简短、可执行，并带 1～2 个示例。

## 与根目录 AGENTS.md 的关系

- **AGENTS.md**：项目总览、目录结构、常用命令、协作约定，适合人类和 AI 一起阅读。
- **`.cursor/rules/*.mdc`**：针对某类文件或某类任务的具体规范，由 Cursor 在匹配时自动加载。

两者可同时使用：AGENTS.md 描述「做什么、怎么跑」，rules 描述「怎么写、按什么规范」。

## 修改与生效

- 编辑 `.mdc` 或 AGENTS.md 后无需重启 Cursor，新对话会使用最新内容。
- 若某条规则不再需要，可将 `alwaysApply` 设为 `false` 或删除/重命名该 `.mdc` 文件。
