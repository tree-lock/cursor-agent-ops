# 高德地图租房推荐与通勤分析

## 项目说明
本项目基于高德地图 MCP 工具，针对深圳市南山区粤兴一道9号香港科技大学产学研大楼周边进行租房推荐与通勤分析。

## 主要内容
- **目标地点**：香港科技大学产学研大楼
- **推荐范围**：周边 1.5km 内的小区
- **分析维度**：步行距离、公交/地铁通勤时间、小区详情
- **输出成果**：[详细推荐报告](./rental_recommendations.md)

## 使用工具
- **user-amap-maps**：用于地点搜索、周边查询、路径规划
- **user-chrome-devtools**：用于验证地图链接跳转有效性

## 关键发现
- **最佳通勤方式**：对于目标地点周边 1-1.5km 的小区，步行或骑行通常比公共交通更快捷。
- **推荐小区**：中信·海阔天空雅居、滨福庭园、阳光带海滨城等。
